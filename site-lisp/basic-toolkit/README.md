# Basic Toolkit
Here include some useful functions for Emacs.
## 1. Open File in External App

``` Elisp
(open-file-in-external-app)
```
This a emacs command to open the current file or marked dired files in external app. (as if you double-clicked the file on desktop) It's useful for image files, PDF file, video, audio files.
I use it for open markdown file in opora to preview.
